# A Data Science–Driven Study of Feature Influence Using Machine Learning and Explainable AI

## Overview
This project focuses on analyzing feature influence in machine learning models using Explainable AI (XAI) techniques. It provides insights into model predictions by identifying the contribution of individual features through SHAP and LIME, improving model transparency and interpretability.

## Features
- Data preprocessing
- Machine learning model training and evaluation
- Feature importance analysis
- Explainable AI using SHAP and LIME
- Visualization of model explanations

## Technologies Used
- Python
- Jupyter Notebook
- Pandas
- NumPy
- Scikit-learn
- Matplotlib
- SHAP
- LIME

## Project Structure

```
README.md

```
